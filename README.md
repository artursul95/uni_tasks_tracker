# uni_tasks_tracker
Удобное графическое приложение для отслеживания задач по дисциплинам.

Чтобы воспользоваться приложением достаточно установить exe файл из дериктории dist.  

Для редактирования используйте команды ниже:  

git clone https://github.com/artursul95/uni_tasks_tracker  

pip install -r requirements.txt
